# ÉLITE PERFORMANCE - Athletecore Pro (姿勢・動作バイオメカニクス分析プラットフォーム)

ジュニアアスリート（10歳〜22歳）向けのスマートフォン姿勢・動作分析アプリと、専門指導者（理学療法士・トレーナー・メンター）向けの分析ポータルを統合した、PWA対応のWebプラットフォームです。

---

## 🌟 主な機能・特徴

### 1. アスリート（エンドユーザー）モード【スマホ推奨・PWA対応】
- **モバイルファースト設計**: スマホでの撮影・測定に適した全画面UI（PWA対応によりホーム画面にアイコン追加して起動可能）。
- **自動姿勢・荷重分析**: 骨格追跡、左右の荷重比、足底圧中心（COP）の重心動揺軌跡を測定。
- **自動スケール比率推定**: マット校正を行わなくても、身長から生物学的に自動でスケール（1pxあたりのcm比率）を概算します。
- **Gemini AI 臨床インサイト**: 測定データに基づき、姿勢の癖や怪我リスク、推奨ストレッチメニューをAIが自動生成。
- **メンター予約機能（有料セッション）**: AI診断で納得いかない場合、登録専門家（理学療法士やトレーナー）の有料個別指導枠をその場で予約可能。

### 2. 専門家・指導者モード【PC 16:9 ワイド推奨・ID/PASSロック付】
- **デモログイン情報**: 右上の「🔒 専門家ログイン」から以下で解除できます。
  - **専門家ID**: `specialist`
  - **パスワード**: `athletecore2026`
- **16:9 ワイドスプリット画面**: 左側に大画面のカメラ/録画再生映像、右側にコントロールパネルと各種グラフカルテを配置。
- **D-pad フレーム修正**: AIが誤検出した特定のフレームの関節ポイントを、十字キー（D-pad）でピクセル単位で手動補正可能。
- **専門家カルテ評価入力**: 専門的な指導者アセスメントと処方リハビリメニューをフォームから入力・保存。

## 🔄 双方向データ連携フロー (SaaSワークフロー)

サーバーデータベースを構築することなく、以下の簡単な手順で「選手 ➔ 専門家 ➔ 選手」のクローズドな個別カルテ連携が行えます。

1. **選手（スマホ）**: 自宅やグランドで姿勢やスクワットを録画測定 ➔ 再生画面から「🎥 動画出力」やデータを保存。
2. **選手（スマホ）**: 履歴からデータを選択し、「📥 CSV」等のデータとともに保存したセッション（またはJSON）を専門家へ共有。
3. **専門家（PC）**: アカウントでロック解除 ➔ コントロールパネルの **「JSON読込」** から選手のデータをインポート。
4. **専門家（PC）**: アライメントのズレを確認し、必要に応じてD-padで骨格を補正 ➔ 「専門家カルテ評価」を入力し **「保存」** をクリック。
5. **専門家（PC）**: **「📤 JSON書出」** をクリックして、カルテ評価付きのセッションファイルを選手に返信。
6. **選手（スマホ）**: 返信されたJSONファイルをアプリ上でインポート ➔ レポートを開くと、GeminiのAI評価の他に **「👩‍⚕️ 担当専門家・メンターによる評価カルテ」** が追加され、個別指導の内容を確認できます！

## 🌐 GitHub Pages へのデプロイ手順

1. ご自身のGitHubアカウントに `connect-ai-v2` という名前のPublicリポジトリを新規作成します。
2. 作成したフォルダ `connect-ai-v2` 内のすべてのファイル（`index.html`, `style.css`, `db.js`, `biomechanics.js`, `api.js`, `app.js`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`）をGitHubのブラウザ画面にドラッグ＆ドロップしてアップロード（コミット）します。
3. リポジトリの **Settings** > **Pages** に移動し、Branchを **`main`** に設定して **Save** します。
4. 数分後、`https://あなたのユーザー名.github.io/connect-ai-v2/` でアプリが全世界にSSL（HTTPS）公開されます。

## 📱 PWA（ホーム画面への追加）方法
GitHub Pagesに公開後、スマホのブラウザ（iOSのSafari、AndroidのChrome）でサイトを開き、**「共有（またはメニュー）」➔「ホーム画面に追加」**を選択します。
ホーム画面にオリジナルのネオンロゴアイコンが追加され、次回から全画面のスタンドアロンアプリとしてワンタップで起動できます。
